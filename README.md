# Masfy Consultants

Masfy Consultants is a leading engineering consultancy firm specializing in civil and structural engineering solutions. With a strong presence in East Africa, we deliver innovative and sustainable engineering services for a wide range of projects, from commercial towers to residential complexes and infrastructure development.

## Features

- Comprehensive civil and structural engineering services
- Sustainable and innovative design solutions
- Advanced 3D modeling and analysis
- Project management and consultancy
- Geotechnical and surveying services

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (v1.0.0 or higher)
- Node.js (v18.0.0 or higher)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/LoftUganda/masfy.git
   cd masfy
   ```

2. Install dependencies:
   ```sh
   bun install
   ```

### Running the Application

1. Start the development server:
   ```sh
   bun run dev
   ```

2. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Environment Variables

Create a `.env` file in the root directory and add the following variables:

```
API_KEY=your_airtable_api_key
BASE_ID=your_airtable_base_id
```

## Project Structure

- `/src` - Source code
  - `/components` - Reusable UI components
  - `/views` - Page components
  - `/lib` - Utility functions and API calls
- `/static` - Static assets (images, CSS, JS)
- `/template` - HTML templates
